# ComputerNetworks-4390-Project

Website to view Yu-Gi-Oh! cards.


Helps users sort cards based on a specific archetype and play-format (ie. edison cards before 2010s, etc.)


# Tech stack

Application is developed using Docker to ease deployement.

Frontend - React, tailwind CSS
Backend - Python


# How to access the application

## Hosted website:

You can acesss the website until January 10th, 2026 at "{{insert website}}"

## Local install:

To locally run this website with a limited dataset:

We will be using the `docker-compose.yml` file to set up the program.

1. `docker compose up -d --build`
